# 粤语闯关厨房（静态原型版）

这是一个无需安装依赖即可运行的粤语学习网站原型，包含：

- 12 个按顺序解锁的学习关卡
- 每关前置学习 + 小测试
- 通关奖励（支持插入奖品图片）
- 通关后可跳转到烹饪界面，使用奖励组合出菜品

## 快速使用

1. 直接双击打开 `index.html`；
2. 或使用任意静态服务器启动后访问。

## 功能说明

### 1) 关卡闯关机制

- 总计 12 关，必须上一关通过后才能进入下一关；
- 每关包含：
  - 一段前置学习内容；
  - 一段标准发音播报（必须完整听完）；
  - 1 道测试题；
  - 1 次语音读音挑战（必须读出目标词句）；
- 只有“听完前置发音 + 答题正确 + 读音挑战通过”才判定通关并解锁奖励。

### 2) 奖品系统

- 每关通关后展示奖励卡片；
- 页面预留了“奖品图片占位区”。

#### 如何插入奖品图片

在 `script.js` 的 `levelData` 中，每个关卡的 `reward.image` 默认是空字符串：

```js
reward: {
  name: "新鲜虾仁",
  desc: "通关奖励：弹牙虾仁，可用于烹饪。",
  image: "", // 改成图片 URL
}
```

你可以改成：

- 网络图片 URL，例如 `https://xxx.com/shrimp.jpg`
- 本地相对路径，例如 `./assets/rewards/shrimp.png`

## 3) 烹饪界面

- 用户可在通关后选择跳转到烹饪界面；
- 目前内置 11 个固定菜谱（菜市场摊位）；
- 每个菜谱会显示所需食材；
- 只有当用户已收集全部对应食材时，菜谱按钮才可点击烹饪。

## 数据持久化

- 使用 `localStorage` 保存当前进度与已解锁奖励；
- 刷新页面后会保留进度；
- 可在页面点击“重置进度”清空数据。

## 语音识别说明

- 使用浏览器 `Web Speech API` 进行实时识别；
- 首次使用请允许网页访问麦克风；
- 若浏览器提示不支持语音识别，请使用最新版 Chrome 或 Edge。
- 前置学习发音使用浏览器语音播报（SpeechSynthesis），同样建议 Chrome/Edge。
